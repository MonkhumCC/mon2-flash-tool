History

FW 1.3

Feb 2024

- Add sampler mode

  • Users can access sampler mode by pressing SHIFT + CT. Cancel the random 8 step function.
  • Up to 12-position, 16-bit, 48000 Hz audio files can be programmed with a total duration of ~40 seconds.

    ** go to website https://monkhumcc.github.io/mon2-wav-convertor/ to upload audio file.
  • Divided into 3 sound groups [A1, A2][B1,B2][C1,C2].
  • Play with OSC1 & adjust param at page 13.

- Add shortcut key at sampler mode

  • shortcut key CLR + A1 to go to the Samp 1 page.
  • shortcut key CLR + A2 to go to the Samp 2 page.
  • shortcut key CLR + B1 to go to the Samp 3 page.
  • shortcut key CLR + B2 to go to the Samp 4 page.
  • shortcut key CLR + C1 to go to the Samp 5 page.
  • shortcut key CLR + C2 to go to the Samp 6 page.
  • ** Press a second time to switch pages between samp name and samp pitch.

- Add shortcut key at voice mode & chord mode

  • shortcut key CLR + A1 to go to the OSC Freqency page.
  • shortcut key CLR + A2 to go to the waveform & volume page.
  • shortcut key CLR + B1 to go to the mod OSC1 & mod OSC2 page.
  • shortcut key CLR + B2 to go to the mod OSC3 & envelope OSC1 page.
  • shortcut key CLR + C1 to go to the envelope OSC2 & envelope OSC3 page.
  • shortcut key CLR + C2 to go to the distortion & filter page.
  • ** Press a second time to switch pages.

- The knob is related to the original parameter value. The knob has 4 states.

  • The knob is locked to prevent the value from being accidentally adjusted.
    [====] belt symbol.
  • The knob is ready to work. When you turn the knob, it will work immediately.
    [ .] dot symbol.
  • The knob is not ready to work. When the knob is turned, no change will occur.
    [ ] nothing symbol.
  • The knob is activated when the knob is turned to a value close to the current value of the original variable.
    [ C] letter C symbol.

- Update new chord table.

  • Adjust the notes in the new chord table to [ "Maj ", "Maj7", "Min ", "Min7", "7th ", " 9 ", "Maj9" ] .

FW 1.2

Oct 2023

update USB MIDI input

  • Added USB MIDI for use with MIDI Note. Users can choose to use MIDI to control an oscillator group by pressing SHIFT +AT. Find the M6/6 page and select the MIDI Channel for each group. And after that, come to Voice Mode by pressing SHIFT + REC.
