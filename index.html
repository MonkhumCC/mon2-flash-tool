History

FW 2.1

Jun 2025

- Added multi-pattern save system (6 slots)

  • Users can save up to 6 separate pattern setups.
  • Each sampler instance has its own independent parameter set.
  • This allows for more flexible and diverse song creation.

- Added wave sampler fade in/out

  • Fade In / Fade Out added to make transitions smoother and more natural.

ไทย:
เพิ่มระบบบันทึก Pattern ได้ 6 แบบ โดย Sampler แต่ละตัวแยก parameter อิสระ
เพิ่มระบบ Fade In / Fade Out เพื่อให้เสียงต่อเนื่องนุ่มนวลขึ้น

FW 2.0

Aug 2024

- Added Step Sequencer Mode

  • Press and hold any button for 3 seconds to enter Step Sequencer mode.

ไทย:
เพิ่มโหมด Step Sequencer
กดปุ่มค้าง 3 วินาทีเพื่อเข้าสู่โหมดนี้

FW 1.3

Feb 2024

- Add Sampler mode

  • Users can access sampler mode by pressing SHIFT + CT. Cancel the random 8 step function.
  • Up to 12-position, 16-bit, 48000 Hz audio files can be programmed with a total duration of ~40 seconds.

    ** go to website https://monkhumcc.github.io/mon2-wav-convertor/ to upload audio file.
  • Divided into 3 sound groups [A1, A2][B1,B2][C1,C2].
  • Play with OSC1 & adjust param at page 13.

- Add shortcut key at sampler mode

  • shortcut key CLR + A1 → Samp 1 page
  • shortcut key CLR + A2 → Samp 2 page
  • shortcut key CLR + B1 → Samp 3 page
  • shortcut key CLR + B2 → Samp 4 page
  • shortcut key CLR + C1 → Samp 5 page
  • shortcut key CLR + C2 → Samp 6 page
  • ** Press a second time to switch pages between samp name and samp pitch.

- Add shortcut key at voice mode & chord mode

  • shortcut key CLR + A1 → OSC Frequency page
  • shortcut key CLR + A2 → waveform & volume page
  • shortcut key CLR + B1 → mod OSC1 & mod OSC2 page
  • shortcut key CLR + B2 → mod OSC3 & envelope OSC1 page
  • shortcut key CLR + C1 → envelope OSC2 & envelope OSC3 page
  • shortcut key CLR + C2 → distortion & filter page
  • ** Press a second time to switch pages.

- The knob is related to the original parameter value. The knob has 4 states.

  • Locked: prevent accidental adjustment [====] belt symbol.
  • Ready: knob works immediately [ . ] dot symbol.
  • Inactive: no change when turned [ ] blank symbol.
  • Catch: activates when near original variable [ C ] letter C symbol.

- Update new chord table.

  • Chord table updated to [ "Maj", "Maj7", "Min", "Min7", "7th", "9", "Maj9" ].

ไทย:
เพิ่มโหมด Sampler พร้อมการควบคุมเสียงสูงสุด 48kHz/16bit ยาว 40 วินาที
เพิ่ม Shortcut Key หลายแบบทั้งใน Sampler และ Voice/Chord mode
เพิ่มระบบจับค่า Knob แบบ 4 สถานะ
อัปเดตตารางคอร์ดใหม่

update USB MIDI input
